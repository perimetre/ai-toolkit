# perimetre-documents

Claude plugin for professional document writing. Covers structure, clarity, tone, and formatting for reports, proposals, and presentations.

## Audience

General team (non-devs and devs) generating documents for clients or internal use.

## Skills

| Skill | Description |
|-------|-------------|
| `document` | Clean, professional document principles — structure, executive summaries, headings, tables, tone |

## Pairing with perimetre-brand-guidelines

This plugin is brand-agnostic by design. Install `perimetre-brand-guidelines` alongside this plugin to add Périmètre colour, typography, and logo guidance on top of the structural principles here.
